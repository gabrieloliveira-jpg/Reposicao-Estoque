ZONA B — Catálogo via IA

Catálogos onde "ESGOTADO" é uma imagem sobreposta (ex: AIKER). Envie o PDF aqui no Claude com o comando abaixo e cole o JSON retornado.

Analise este catálogo visualmente página por página e retorne APENAS um JSON válido, sem texto adicional, sem explicações, sem markdown. Siga exatamente este formato: {"produtos":[{"sku":"SKU-AQUI","disponivel":true,"pcsCx":0}]} Regras: - SKU com imagem + descrição = disponivel true - Espaço com selo ESGOTADO/SEM ESTOQUE/similar = disponivel false - pcsCx = número após PCS/CX, ou null se não encontrar - Ignore cabeçalhos, rodapés, preços e categorias - Retorne apenas o JSON, nada mais
